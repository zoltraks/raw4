```
8-bit to 4-bit data conversion tool
Author: Filip Golewski
Link: http://github.com/zoltraks/raw4.git

Options:

  -i <source>
  --input <source>

     Source file for reading

  -o <destination>
  --output <destination>

     Destination file for writing

  -v
  --verbose

     Verbose mode

  -f
  --force

     Force destination file overwrite if exists

  -r
  --reverse

     Reverse output data nibble order

  -a
  --average

     Round input value instead of ceiling
```
